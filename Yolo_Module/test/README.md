# 鼠标点击生成敌人 - 快速启动

## 使用方法

### 一键启动

```bash
python start_click_sim.py
```

## 功能说明

启动后：
1. ✅ 仿真器自动打开
2. ✅ 显示鼠标十字准星
3. ✅ 实时显示坐标
4. ✅ 点击地图任意位置生成红色敌人
5. ✅ 按 `C` 清除所有敌人
6. ✅ 按 `ESC` 退出

## 操作说明

```
鼠标操作:
  • 移动鼠标 - 显示十字准星和坐标
  • 左键点击 - 在点击位置生成敌人

键盘操作:
  • 按 C - 清除所有敌人
  • 按 ESC - 退出仿真器
```

- ✅ **精确位置** - 在鼠标点击位置精确生成
